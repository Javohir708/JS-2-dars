JavaScript

JavaScript 1995-yilda Netscape kompaniyasi tomonidan ishlab chiqilgan. Uni Brendan Eich yaratgan va dastlab "Mocha" deb nomlangan. Keyinchalik "LiveScript" deb o‘zgartirilgan, ammo Netscape brauzerining yuksalish davrida Java dasturlash tilidan ilhomlanib, til nomini "JavaScript" deb o‘zgartirishgan. 1996-yilda JavaScript ECMAScript standarti bilan rasmiylashtirilib, bugungi kunda veb-saytlar uchun asosiy dasturlash tillaridan biriga aylangan. JavaScript saytlarni dinamik qilish uchun ishlatiladigan asosiy dasturlash tilidir. Saytning dinamikligi deganda, u sahifa yangilanmasdan o‘zgarishlar qilish, foydalanuvchi bilan real vaqtda muloqot qilish va kontentni avtomatik ravishda yangilash imkoniyatlari tushuniladi. JavaScript orqali foydalanuvchi tugmalarni bosganda oynalar ochilishi, shakllardagi ma’lumotlarning tekshirilishi yoki sahifadagi animatsiyalarni boshqarish kabi interaktivlik qo‘shish imkoniyatiga ega bo'ladi.

JavaScript Tarixi va Yangilanishlar

JavaScript dastlabki chiqarilishi 1995-yilda Brendan Eich (Netscape) tomonidan ishlab chiqarilgan. 1997-yilda ES1 versiyasi chiqariladi. Shundan keyin yangi versiyalari ishlab chiqarila boshlaydi:

  • 1998 - ES2: Kichik o'zgarishlar va standartni yaxshilash.
  • 1999 - ES3: JavaScript'ni kengaytiradigan xususiyatlar kiritildi, masalan, yaxshilangan qatorlar, xatolarni tutish va boshqa metodlar.
  • 2009 - ES5: JavaScript'ning eng mashhur versiyalaridan biri. 'Strict mode', JSON qo'llab-quvvatlovi, va yangi metodlar bilan tanish bo'ldi. Bu versiya keng tarqaldi va zamonaviy veb dasturlashda asosiy asos bo'lib xizmat qildi.
  • 2015 - ES6 (ES2015): Katta o'zgarishlar, jumladan, let/const, arrow functions, classlar, va modullar kiritildi. Bu versiya JavaScript'ni boshqa zamonaviy dasturlash tillari darajasiga ko'tarishga yordam berdi...

Muvaffaqiyatlar

  • 1. Keng Qamrovli Foydalanish: JavaScript veb dasturlash, mobil dasturlar, server tomonidagi dasturlar va hatto ish stoli ilovalarida keng qo'llaniladi.
  • 2. Katta Jamoa: JavaScript rivojlanishi va qo'llab-quvvatlanishiga katta jamoa hissa qo'shmoqda, turli kutubxonalar va ramkalar (masalan, React, Angular, Vue.js) ishlab chiqilmoqda.
  • 3. Doimiy Yangilanishlar: ECMAScript standartlari muntazam ravishda yangilanib, yangi xususiyatlar va sintaksis qo'shiladi. Bu dasturchilarga yanada samarali va qulay ishlash imkonini beradi.
  • 4. Muvaffaqiyatli Kutilgan Tizimlar: Node.js, Express.js va boshqa server tomonidagi texnologiyalar JavaScriptni backend dasturlashda mashhur qildi.

console.log()

JavaScriptdagi console obyekti brauzer yoki ishlash muhiti bilan o'zaro aloqada bo'lish uchun ishlatiladi, ayniqsa xatolarni izlash, debugging va hodisalarni qayd qilishda yordam beradi. Bu o'zingiz yaratgan kodlar haqida ma'lumot olish uchun juda qulay vosita hisoblanadi

console.log()

Kodlarni test qilish va umumiy ma'lumotlarni chop etish uchun ishlatiladi. Bu kodda turli o'zgaruvchi yoki xabarlarni chop etadi.

console.log("Hello, World!");

console.log(25 + 5); // Natija: 30

console.error()

Xatolik yuz berganda xabar chop etish uchun ishlatiladi. Ko'pincha qizil rangda chiqadi va xatolarni tezroq ko'rishga yordam beradi.

console.error("Xatolik yuz berdi!");

console.warn()

Ogohlantirishlarni chop etadi, ammo bu xatolik emas. Kodda potentsial xato bo'lishi mumkin bo'lgan joylarni ko'rsatish uchun ishlatiladi.

console.warn("Bu faqat ogohlantirish.");

console.info()

Ma'lumot sifatida xabar chop etadi. Bu console.log() ga o'xshaydi, lekin ko'pincha faqat ma'lumot uchun ishlatiladi.

console.info("Bu umumiy ma'lumot.");

console.assert()

Berilgan shart noto'g'ri bo'lsa, xatolik xabarini ko'rsatadi. Shart to'g'ri bo'lsa, hech narsa chiqarmaydi.

console.assert(5 > 10, "Bu xabar noto'g'ri bo'lsa chiqadi.");

console.clear()

Konsolni tozalaydi, faqat konsol oynasida bo'sh joy qoldiradi.

console.clear();

Variable

JavaScriptda o'zgaruvchilar (variables) ma'lumotlarni saqlash uchun ishlatiladi. O'zgaruvchilar yordamida qiymatlarni o'zgartirish va boshqarish mumkin.

  • 1. var: Eski versiyalarda o'zgaruvchilarni e'lon qilish uchun ishlatilgan. O'zgaruvchilarni global yoki funksional doirada yaratadi. Qayta e'lon qilish va qiymatini o'zgartirish mumkin.

  • var a = 10;

    a = 20; // a ning yangi qiymati 20

  • 2. let: ES6 (ECMAScript 2015) dan boshlab kiritilgan. Blok doirasida o'zgaruvchilarni yaratadi, ya'ni faqat e'lon qilingan blok ichida mavjud. Qayta e'lon qilish mumkin emas, ammo qiymatini o'zgartirish mumkin.

  • let b = 30;

    b = 40; // b ning yangi qiymati 40

  • 3. const: ES6 dan kiritilgan. O'zgaruvchi qiymatini bir marta e'lon qilish mumkin va keyinchalik o'zgartirib bo'lmaydi (lekin agar u ob'ekt yoki massiv bo'lsa, uning ichki qiymatlari o'zgartirilishi mumkin).

  • const c = 50;

    // c = 60; // Xato: const o'zgaruvchini o'zgartirib bo'lmaydi
