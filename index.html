JavaScript

JavaScript 1995-yilda Netscape kompaniyasi tomonidan ishlab chiqilgan. Uni Brendan Eich yaratgan va dastlab "Mocha" deb nomlangan. Keyinchalik "LiveScript" deb o‘zgartirilgan, ammo Netscape brauzerining yuksalish davrida Java dasturlash tilidan ilhomlanib, til nomini "JavaScript" deb o‘zgartirishgan. 1996-yilda JavaScript ECMAScript standarti bilan rasmiylashtirilib, bugungi kunda veb-saytlar uchun asosiy dasturlash tillaridan biriga aylangan. JavaScript saytlarni dinamik qilish uchun ishlatiladigan asosiy dasturlash tilidir. Saytning dinamikligi deganda, u sahifa yangilanmasdan o‘zgarishlar qilish, foydalanuvchi bilan real vaqtda muloqot qilish va kontentni avtomatik ravishda yangilash imkoniyatlari tushuniladi. JavaScript orqali foydalanuvchi tugmalarni bosganda oynalar ochilishi, shakllardagi ma’lumotlarning tekshirilishi yoki sahifadagi animatsiyalarni boshqarish kabi interaktivlik qo‘shish imkoniyatiga ega bo'ladi.
